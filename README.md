# README.md

## Prerequisites
Node.js version >= 18

## Getting started

### Installation
```bash
npm install
```

### Running project
```bash
npm run dev
```

### Tech stack:
* [Next.js](https://nextjs.org/)
* [TailwindCSS](https://tailwindcss.com/)
* [Shadcn](https://ui.shadcn.com/)
* [Tanstack/table](https://tanstack.com/table/latest)
